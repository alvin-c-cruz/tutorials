Variables

Variables are data containers. Its like a bucket where you put something for later use. To create a variable simply assign a value to it, no declaration needed. The following code will print number 39.

    my_var = 39
    print(my_var)

Data Type

The basic variable data types are:

  1. String
  2. Number
  3. Boolean
  4. Tuple
  5. List
  6. Dictionary

String

String should be surrounded by either single quotation marks, or double quotation marks.

    my_string = 'this is a string'
    another_string = "this is also a string"

    print(my_string)
    print(another_string)

Number

Python has three number data types, int, float and complex.

    my_int = 10
    my_float = 25.75
    my_complex = 1j

    print(my_int)
    print(my_float)
    print(my_complex)

Boolean

Boolean represents either True or False.

    my_bool = True
    print(my_bool)

Tuple

Tuple stores multiple variables separated by comma in a single variable. It is enclosed by brackets. Once the tuple is created the contents cannot change. 

    my_tuple = (1, "two", 3, "4")
    print(my_tuple)

List

List stores multiple variables separated by comma in a single variable. It is enclosed by square brackets. The contents of a list can change.

    my_list = [1, "two", 3, "4"]
    print(my_list)

Dictionary

Dictionary uses key and value pairs in storing multiple variables. It is enclosed by curly brackets. The contents of a dictionary can change.

    my_dict = {"key_1": "value_1", "key_2": "value_2", 3: "value_3 with key of type int"}
    print(my_dict)
