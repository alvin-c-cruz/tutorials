Running Python Scripts

There are three ways to run a python script.

  • Using python's IDLE Shell
  • Via the command prompt
  • From the built-in terminal of IDE

In this tutorial we will create a simple python file that prints Hello World!. The code is as follows:

print("Hello World!")

Using python's IDLE Shell

IDLE Shell is installed together with python. You can write codes with idle and run it in the shell.

  1. Create a folder that will contain the python file
  2. Create a file inside the folder and change the extension name to .py
  3. Right-click the .py file and choose Edit with IDLE
  4. Type print("Hello World!") in the .py file
  5. Choose Run in IDLE's menu bar and click Run Module. The shortcut key is F5.

The python code will run in IDLE Shell.
